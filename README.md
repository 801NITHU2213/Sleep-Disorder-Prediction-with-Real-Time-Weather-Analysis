# Sleep Disorder Prediction using Decision Tree (CART)

This project builds a CART decision tree model to predict sleep disorders using
the **Sleep Health and Lifestyle Dataset**.

## Features
- Data preprocessing
- Train–test split (80-20)
- CART model using `rpart`
- Tree visualization
- Confusion matrix evaluation
- ggplot2 heatmap visualization

## Files
- `sleep_disorder_decision_tree.R` — Main R script
